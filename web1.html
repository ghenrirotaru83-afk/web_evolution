Web 1.0: The Static Web

(1991-2004)

Key Characteristics

  • Read-only web with static HTML pages that rarely changed
  • Limited user interaction, typically just clicking hyperlinks
  • Unidirectional information flow from private web servers to browsers
  • Webmasters acted as gatekeepers of content
